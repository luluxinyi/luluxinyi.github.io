Whim

Designing the world's first MaaS solution

Background

Whim is an app that combines all your journeys with public transport, taxi, car, bikes and more in one service.

My role

  • Research & Wireframing
  • UI/UX Design
  • Visual Design
  • Prototyping

Various UI designs

While working on the project, we refreshed the visual style, introduced new features, as well as designed features from the roadmap that will be implemented later on the service.

Tickets

We designed generic mobile tickets to be used by various service providers in different countries.

By using an animated background, 2-digit verification code and QR code in the inspection view, we made mobile tickets secure and hard to replicate.
